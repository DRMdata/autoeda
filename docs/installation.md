# Installation

## Stable release

To install AutoEDA, run this command in your terminal:

```sh
uv add autoeda
```

Or if you prefer to use `pip`:

```sh
pip install autoeda
```

## From source

The source files for AutoEDA can be downloaded from the [Github repo](https://github.com/DRMdata/autoeda).

You can either clone the public repository:

```sh
git clone git://github.com/DRMdata/autoeda
```

Or download the [tarball](https://github.com/DRMdata/autoeda/tarball/master):

```sh
curl -OJL https://github.com/DRMdata/autoeda/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd autoeda
uv pip install .
```
